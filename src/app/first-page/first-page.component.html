WHAT IS BEELINE ?

One day, we had a very important meeting to go to.And we did not want to be late. So we wanted to make sure that the route we are heading to isn't busy at the moment. We searched everywhere but found nothing to help us know So we, as software engineers , decided to create it !

BEELINE : a web application that makes driving easier. It allows drivers to know the state of traffic on a certain route and recommends an alternative if needed.
